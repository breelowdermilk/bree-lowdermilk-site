# Sitemap & Information Architecture
- **Home** (/)
- **Songs** (/songs) — index with filters & search
  - **Song detail** (/songs/{slug})
- **Works** (/works) — albums/recordings & shows/projects
  - **Album/Recording** (/works/recordings/{slug})
  - **Show/Project** (/works/shows/{slug})
- **Events** (/events)
  - **Event detail** (/events/{slug})
- **EPK / Press** (/press)
- **News** (/news) — optional announcements
- **Contact** (/contact)
- **Sheet Music** (/sheet-music) — overview + outbound to providers

## Cross‑linking Rules
- Song → Show/Album → related songs.
- Song → Sheet‑music external link (new tab).
- Event → related works/songs setlist (optional).
- Press quotes appear on Home (marquee), Works, and EPK.
