# Glossary
- **Astro**: static site builder with partial hydration ("islands").
- **AstroWind**: free starter theme for Astro.
- **MDX**: Markdown + JSX.
- **Pagefind**: static search indexer & UI.
- **JSON‑LD**: machine‑readable SEO data format.
- **WCAG**: web accessibility guidelines.
- **Core Web Vitals**: Google performance metrics (LCP/INP/CLS).
