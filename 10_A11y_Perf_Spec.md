# Accessibility & Performance
## WCAG 2.2 AA must-haves
- Keyboard navigable menus & modals; visible focus indicators.
- Target size >= 24px; touch spacing generous.
- Color contrast AA; respect `prefers-reduced-motion`.
- Forms labeled; error states announced.
