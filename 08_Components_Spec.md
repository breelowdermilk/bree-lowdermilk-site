# Components Spec
- **SongCard**: title, show, voice type pills, key list, sheet‑music button, quick‑play embed (optional).
- **SongFilters**: checkbox groups (voice type, mood), select (show), text search (Pagefind input).
- **SongDetail**: hero (title + show), key/range panel, media embeds, lyrics accordion, credits, sheet‑music CTA.
- **EventList**: date, city/venue, tickets link; iCal/Google add‑to‑calendar.
- **PressQuote**: quote + outlet + link; large on home hero strip.
- **YouTube**: lazy‑loaded iframe with title & captions support.
- **SpotifyEmbed**: iframe with reduced motion respect.
- **Navbar**: skip link, focus ring, reduced‑motion respectful underline animations.
- **Footer**: socials, newsletter capture, rep/management contacts.
